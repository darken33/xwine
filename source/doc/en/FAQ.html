Installation problems

Q1) The next message is diplayed durring the Make step ?
ld: -ldb1 : file not found

R1) The db1 library was not found, please install it

Q2) The next message is diplayed durring the Configure step ?
gnomeConf.sh : file not found

R2) The gnome-libs-devel package is not installed, please install it

Running problem

Q1) The next message is displayed when you run XWine ?
** WARNING ** : Couldn't find pixmap file: xwine/winfd.xpm

R1) The default XWine installation directory is /usr/local two solutions:
  • Restart installation with ./configure --prefix=/usr
  • Make the next link: ln -s /usr/local/share/pixmaps/xwine /usr/share/pixmaps/xwine
Q2) The next message is diplayed in the log file ?
wine: file '/home/philippe/.wine/wineserver-localhost.localdomain/socket' exists,
but I cannot connect to it; maybe the wineserver has crashed?
If this is the case, you should remove this socket file and try again.

R2) A software is crashed, you have to flush the wineserver socket: click on Programs>Reset
