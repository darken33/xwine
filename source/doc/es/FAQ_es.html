Problemas de instalación

P1) ¿El siguiente mensaje se muestra durante el paso Make?
ld: -ldb1 : file not found

R1) La librería db1 no se encontró, por favor instalala.

P2) ¿El siguiente mensaje se muestra durante el paso Configure?
gnomeConf.sh : file not found

R2) El paquete gnome-libs-devel no está instalado, debes instalarlo.

Problemas de ejecución

P1) ¿El siguiente mensaje se muestra cuando ejecutas XWine?
** WARNING ** : Couldn't find pixmap file: xwine/winfd.xpm

R1) El directorio de instalación por defecto de XWine es /usr/local . Dos soluciones:
  • Reinicia la instalación con ./configure --prefix=/usr
  • Haz el siguiente enlace: ln -s /usr/local/share/pixmaps/xwine /usr/share/pixmaps/xwine
P2) ¿El siguiente mensaje se muestra en el fichero log?
wine: file '/home/philippe/.wine/wineserver-localhost.localdomain/socket' exists,
but I cannot connect to it; maybe the wineserver has crashed?
If this is the case, you should remove this socket file and try again.

R2) Una aplicación ha fallado, tienes que matar el socket del servidor Wine: haz click en Programas>Reiniciar
