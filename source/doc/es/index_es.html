Introducción

  XWine es un interfaz gráfico de usuario para el emulador Wine.

Con XWine puedes configurar y ejecutar aplicaciones de MS-Windows (Ms Dos, Windows 3.x o Win32).

Funciones de XWine:
  • Configuración de Wine
  • Configuración y gestión de aplicaciones de MS-Windows

Instalación

  Librerías que necesitas:
  • GTK
  • Gnome
  • db1

Software que necesitas:
  • Wine : XWine  es un interfaz gráfico de usuario para Wine ;-)
  • Un navegador WEB: es tu navegador para obtener ayuda

Instalación de XWine
  1. Entra en el directorio de XWine:
    $ cd XWine-x.x (x.x es la versión de XWine)
  2. Lanza la configuración de XWine:
    $ ./configure
  3. Lanza la compilación de XWine:
    $ make
  4. Tienes que abrir una sesión como root:
    $ su
  5. Ejecuta la instalación de XWine:
    # make install
Para más información puedes leer el fichero INSTALL incluído en el archivo XWine-x.x.tar.gz.

Ejecutar XWine
  • Escribe el siguiente comando:
    $ xwine

Configuración de Wine

  XWine permite configurar Wine a través de menús gráficos:

Versión de Windows
Aquí se puede especificar la versión de Windows a emular.

Configuración de Unidades de disco
Aquí se pueden añadir, modificar o borrar unidades de disco para Wine.

Directorios del Sistema
Aquí se definen dónde se encuentran los directorios de Windows. También se define la variable PATH.

Redefinir DLLs
Aquí se definen cuáles DLL quieres usar (DLL de Windows o DLL de Wine).

Interfaz de usuario
Aquí se puede configurar el aspecto de Wine (Windows 3.1, 95 o 98). También se puede configurar las fuentes del sistema.

Configuración de X11
Aquí se puede configurar si la aplicación de Wine tiene que lanzarse en un escritorio, o ser gestionada o no por las XFree. También se puede configurar el Direct Draw.

Configuración de Puertos
Aquí se pueden configurar los puertos paralelo y de serie. También se puede configurar la impresora.

Configuración del Registro
Aquí se configura dónde tiene que encontrar Wine las informaciones del registro de Windows.

Configuración del Terminal
Se puede especificar el programa de terminal que usará Wine.

Configuración del multimedia
Aquí se especifican los drivers de sonido, onda y midi.

Ejecutar winesetuptk
Si winestuptk está instalado, se puede ejecutar desde este menú.

Fichero de configuración
Aquí también se puede editar la configuración de Wine.

Configurar una aplicación

Configurar una aplicación
  1. Ve a la ventana de Gestión de Aplicaciones:
    haz clic en Inicio>Configuración>Instalar
  2. Para crear, modificar o borrar la configuración de una aplicación:
    haz clic en Nuevo si queremos configurar una nueva aplicación.
  3. Configuración de la aplicación:
    Título : Título de la aplicación
    Ejecutar : Aplicación a lanzar (se puede seleccionar con el explorador: botón "...")
    Usar un CD-ROM : Si está seleccionado,  XWine pedirá que se inserteel CD-ROM antes de ejecutar la aplicación.
    Necesita parámetros : Si la aplicación necesita parámetros.
    Versión de Windows : Se puede seleccionar la versión de Windows (win95,winNT,...)
    Emular MS-DOS : Si se trata de una aplicación de MS-DOS.
    Icono : Se puede seleccionar un icono
    4. Con DLLs y X11drv se puede configurar Wine para una aplicación específica:
    - Redefinir configuración de DLLs
    - Configuración X11
  4. Ahora hay que guardar la configuración:
    Haz click en Aceptar
  5. Ya se puede ejecutar la aplicación:
    haz click en Inicio>Programas>Lista
  6. También se puede crear un icono en los escritorios KDE y GNOME:
    haz click en Inicio>Configuración>Instalar y después en <Escritorio>

Instalar una aplicación
  1. Ejecuta el programa de instalación en el CD-Rom:
    haz click en Inicio>Ejecutar
  2. Cuando la aplicación se ha instalado, es necesario verificar si funciona.  haz clic  en Inicio>Ejecutar
  3. Configurar la aplicación:
    haz click en Inicio>Configuración>Instalar
  4. Verificar la configuración:
    haz click en Inicio>Programas>Lista

Ejecutar una aplicación

  Ejecutar una aplicación que está configurada
  1. Ve al listado de aplicaciones:
    haz click en Inicio>Programas>Lista
  2. Selecciona la aplicación
  3. Lánzala:
    haz click en Ejecutar

Ejecutar otras aplicaciones
Si quieres ejecutar un programa de instalación o verificar si una aplicación funciona antes de configurarla:
  1. Abre la ventana "Ejecutar":
    haz click en Inicio>Ejecutar
  2. Escribe el comando a ejecutar o seleccionalo con el explorador:
    haz click en el botón "..." para mostrar el explorador
  3. Lánzalo:
    haz click en el botón ACEPTAR

Gestionando las aplicaciones

  El fichero log
  Cada cosa que realiza XWine se escribe en un fichero log, que puedes ver haciendo:
Inicio>Configuración>Ver fichero log

Contenidos del fichero:
  • Todas las cosas que realiza XWine
  • Todos los mensajes del emulador Wine.
Cuando cierras XWine puedes guardar o borrar el fichero log.

When you have some problems you should see this file it can help you.
You can print the log file : clic on button <Print>.

La barra de tareas
  Se puede parar un programa que no está funcionando bien haciendo click en  Matar.
El botón Actualizar se usa para actualizar la barra de tareas.

El botón de reiniciar
  Cuando un programa falla puede bloquear el socket del servidor Wine, pero puedes destruirlo haciendo lo siguiente:
Inicio>Programas>Reiniciar
Cuando mates una aplicación, debes usarlo.
