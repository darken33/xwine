7.1. Où trouver XWine ?

XWine va sûrement évoluer, donc si vous voulez récupérer les nouvelles versions,  ou tout simplement vous tenir informé sur ce projet, il est hébergé sur mon site WEB (http://darken33.free.fr) à l'adresse suivante : http://darken33.free.fr/index.php?cat=2&rub=21
