Problèmes d'Installation

Q1) Le message suivant apparaît lors de l'étape Make ?
ld: -ldb1 : file not found

R1) La bibliothèque db1 n'a pas été trouvée, veuillez l'installer

Q2) Le message suivant apparaît lors de l'étape configure ?
gnomeConf.sh : file not found

R2) Le package gnome-libs-devel est manquant, veuillez l'installer

Problèmes d'Utilisation

Q1) Le message suivant apparaît lors du lancement de XWine ?
** WARNING ** : Couldn't find pixmap file: xwine/winfd.xpm

R1) Le répertoire par défaut d'installation de XWine est /usr/local, deux solutions s'offrent à vous:
  • Relancer l'installation avec ./configure --prefix=/usr
  • Faire le lien suivant: ln -s /usr/local/share/pixmaps/xwine /usr/share/pixmaps/xwine
Q2) Le message suivant apparaît dans la log ?
wine: file '/home/philippe/.wine/wineserver-localhost.localdomain/socket' exists,
but I cannot connect to it; maybe the wineserver has crashed?
If this is the case, you should remove this socket file and try again.

R2) Une application s'est mal terminée, il faut supprimer la socket: cliquez sur Programmes>Reset
