2.1. Configuration requise

Bibliothèques nécessaires :

  • GTK : http://www.gtk.org
  • Gnome & Gnome-libs : http://developper.gnome.org
  • La bibliothèque libxml2 : http://rpmfind.net/linux/RPM
  • db1 : http://rpmfind.net/linux/RPM
Remarque :
Si vous désirez installer XWine à partir des sources vous aurez aussi besoin des fichiers d'en-têtes (includes) des bibliothèques. Notamment le package gnome-libs-devel.

Logiciels nécessaires :
Pour que XWine fonctionne, vous aurez également besoin de plusieurs logiciels :

  • Wine : XWine est une interface graphique de configuration pour Wine (ce n'est pas l'émulateur) http://www.winehq.org
  • Un navigateur web : pour lire la documentation (Konqueror, Netscape, Mozilla, Opera)
  • Wine Utils : Ce sont des utilitaires pour Wine tels que bloc-notes, démineur, ...(ce n'est pas obligatoire) http://www.winehq.org
  • WinesetupTk : autre interface graphique de configuration pour Wine, ce logiciel n'est pas obligatoire

2.2. Installation à partir des sources

Les sources se trouvent dans le fichier xwine-1.0.tar.gz (archive), il faut tout d'abord le décompresser :
$ tar -xzf xwine-1.0.tar.gz

Il crée alors un répertoire xwine-1.0 il faut y entrer :
$ cd xwine-1.0_fr

Il faut maintenant configurer l'installation de XWine :
$ ./configure

Maintenant il faut compiler XWine, cela se fait via la commande make :
$ make

Une fois la compilation terminée il suffit de l'installer dans les bons répertoires, pour pouvoir effectuer cette action il faut que vous soyez connecté en tant qu'administrateur (ou root) :
$ su
password:
#

L'installation se fait simplement par la commande make install :
# make install

N'oubliez pas de vous déconnecter du mode administrateur :
# exit
$

Vous pouvez alors exécuter XWine, il suffit de tapper la commande xwine :
$ xwine

2.3. Installation à partir du fichier RPM

L'avantage d'installer XWine par cette méthode est que l'installation est automatiquement gérée par rpm (dépendances, fichiers, désinstallation). De plus vous n'êtes plus obligé de le compiler et donc de posséder les fichiers en-têtes des bibliothèques (les fichiers .h)

Pour installer le programme a partir du fichier RPM xwine-1.0-1.i586.rpm (par exemple) il suffit de taper la commande (en tant que root) :
# rpm -i xwine-1.0.-1.i586.rpm

Et pour le désinstaller il suffit de taper la commande :
# rpm -e xwine
